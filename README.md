## 项目简介​
￼
本项目是一款复刻小兔鲜核心风格的企业级电商实战项目，参考CSDN小兔鲜项目指导规范，聚焦前端实战能力提升，专为刚掌握HTML、CSS、JavaScript原生三件套，或熟悉Vue3技术栈的新手打造，是理论转实践、巩固基础的优质练习场。​
项目以美食/生鲜电商为主题，模拟完整电商业务闭环，一套代码适配微信小程序、H5、App三端运行，无需复杂环境配置，克隆即可上手开发，既可以作为前端新手的实战项目补充简历，也可作为教学案例、个人练手素材，开源免费，支持自由改编、二次开发和商用（遵循MIT协议）。​
核心定位：新手友好、实战导向、多端适配、代码规范，助力开发者快速掌握电商项目开发逻辑，提升前端实战能力和项目架构思维。​
## 技术栈详情​
￼
参考小兔鲜项目技术选型，兼顾基础巩固和企业实战需求，技术栈清晰易懂，无过度复杂依赖，具体如下：​
•
前端核心：Vue3 + TypeScript（类型安全护航，减少开发错误）​
•
框架支撑：UniApp（一套代码多端运行，适配H5、微信小程序、App）​
•
状态管理：Pinia（替代Vuex，轻量高效，适配Vue3组合式API）​
•
UI组件：Uni UI / Element Plus（丰富组件库，加速开发，提升页面美观度）​
•
数据交互：Axios（封装请求拦截、响应拦截，处理接口通信）​
•
编码规范：ESLint + Prettier + Git Hooks（统一代码风格，避免混乱）​
•
基础支撑：HTML5 + CSS3（语义化布局、响应式设计，适配不同设备）​
•
交互实现：JavaScript（原生交互逻辑，如表单验证、页面跳转、动态效果）​
•
其他工具：Git（版本控制）、Vite（快速构建，提升开发效率）​
## 核心功能模块​
￼
复刻小兔鲜电商核心业务流程，覆盖从用户进入页面到完成购物的全场景，模块划分清晰，低耦合易维护，具体功能如下：​
1.
首页模块：轮播图展示、热门商品推荐、分类导航、限时活动展示，优化用户第一眼体验，提升商品曝光率；​
2.
商品模块：商品分类（一级/二级分类）、商品搜索、商品详情（规格参数、库存、评价）、商品收藏，助力用户快速找到目标商品；​
3.
购物车模块：商品添加/删除/修改数量、全选/反选、价格实时计算、商品结算，模拟真实购物车交互逻辑；​
4.
订单模块：订单创建、订单列表、订单详情、订单状态跟踪（待付款/待发货/已完成）、售后服务申请，覆盖订单全生命周期；​
5.
支付模块：集成微信支付接口，模拟支付流程，确保交易安全可靠，同时支持支付状态同步；​
6.
用户模块：微信一键登录（免密码登录）、个人信息编辑、地址管理、收藏夹、订单历史查询，提升用户粘性；​
7.
基础功能：表单验证（登录、注册）、页面路由守卫、错误处理、加载状态提示，优化用户体验。​

## 开发规范​
￼
•
代码风格：遵循ESLint+Prettier规范，禁止冗余代码、未使用变量，注释清晰（关键逻辑必须加注释）；​
•
分支管理：master/main分支为稳定版本，开发新功能请创建feature/*分支，修复bug请创建fix/*分支；​
•
提交规范：提交代码时，commit信息需清晰明了，格式为「类型: 描述」，例如「feat: 新增商品收藏功能」「fix: 修复购物车价格计算错误」；​
•
组件开发：公共组件统一放在components目录下，命名规范为xxx.vue（小写字母+连字符），组件内部逻辑清晰，尽量复用。
